# MyTour 

### 학습 방식

- React 기반으로 직접 UI를 구현하며 웹 애플리케이션 구조 이해
- 기능 단위로 컴포넌트를 분리하여 개발
- Firebase를 활용해 인증과 데이터 관리를 직접 구현

---

### 학습출처

학교 수업에서 학습한 내용을 기반으로, 기능을 확장하여 직접 설계 및 구현한 프로젝트입니다.

---

## 주요 기능

- Firebase 로그인 / 회원 인증
- 여행 기록 조회 / 추가 / 수정 / 삭제
- 좋아요 기능
- 버킷리스트 관리

---

### 사용 기술

- React
- Firebase
- Vite
